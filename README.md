서비스 소개
1. 서비스 개요

현대인은 빠르게 지나가는 하루 속에서 자신의 감정을 제대로 마주할 시간을 가지기 어렵다.
감정 우체통은 사용자가 하루 동안 느낀 감정을 ‘비밀 편지’처럼 작성하면,
AI가 편지 내용을 분석하여 사용자의 감정 상태에 맞는 위로의 답장,
그리고 오늘의 감정에 맞춘 음악·도서 추천, 순간 감정 기록 기능, 캘린더 기반 감정 히스토리를 제공하는 감정 관리 웹 서비스이다.

사용자는 서비스 하나만으로

- 감정 기록

- 감정 분석
  
- 맞춤 피드백

- 힐링 콘텐츠 추천

- 감정 히스토리 관리
  
를 모두 경험할 수 있다.

2. 타 서비스와의 차별점

🔸 1) 단순 감정 분석이 아닌 ‘답장 기반 위로 경험’

다른 서비스들은 감정을 숫자로 표시하거나 태그만 제공하지만,
이 서비스는 “나에게 편지가 도착한 것처럼” AI가 자연스러운 말투로 감정에 공감해 주며 답장을 준다.

🔸 2) 순간 감정 기록 + 캘린더 기반 감정 히스토리

감정을 일기처럼 길게 기록하지 않아도, 버튼 하나로 감정을 빠르게 기록할 수 있다.
캘린더 화면에서 월별 감정의 변화와 패턴을 한눈에 볼 수 있다.

🔸 3) 감정 맞춤형 음악·도서 콘텐츠 연결

‘슬픔’, ‘불안’, ‘기쁨’, ‘지침’ 등의 감정에 기반한
AI 큐레이션 음악/도서 추천을 제공한다.

🔸 4) UX 중심의 ‘정서적 안정’ 디자인

감정 케어를 목표로 했기 때문에,
자극적인 UI 대신 편안한 색감·여백·운동감으로 스트레스 해소 UX를 의도적으로 설계했다.
